<strong>Secure Access Service Edge (SASE)</strong> is a single [[Cloud Computing Glossary#cloud-based|cloud-based]] platform that <mark style="background: #FFB8EBA6;">combines comprehensive WAN capabilities including SD-WAN, routing, and WAN optimization</mark> with <strong>cloud-delivered security services</strong> or SSE (Security Service Edge) such as <strong>SWG (secure web gateway)</strong>, <strong>CASB (cloud access security broker)</strong>, next-generation firewall (<strong>NGFW</strong>) services, and <strong>ZTNA (zero trust network access)</strong> into a [[Networking Glossary#software-defined network|software-defined network]] infrastructure <mark style="background: #BBFABBA6;">for mobile users and remote networks</mark>.

SASE supports branch offices, remote workers, and on-premises secure-access use cases. 
SASE is primarily delivered as a service and enables zero trust access based on the identity of the device or entity, combined with real-time context and security and compliance policies.
